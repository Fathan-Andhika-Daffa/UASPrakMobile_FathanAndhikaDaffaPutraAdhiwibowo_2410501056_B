# EndemikDB
Proyek ini adalah aplikasi ensiklopedia digital berbasis Android (Java) yang menyajikan data flora dan fauna endemik Indonesia. Dibuat khusus sebagai tugas Ujian Akhir Semester (UAS) untuk praktikum Pemrograman Mobile.

# Fitur Utama
**1. Detail Taksonomi Lengkap:** Saya telah menambahkan data Famili dan Genus ke dalam sistem. Sekarang halaman Detail menampilkan informasi taksonomi ilmiah yang lengkap.  
**2. Badge Visual Elegan:** Saya menambahkan "badges" (label berwarna) untuk Famili dan Genus di halaman Detail agar terlihat modern dan rapi.  
**3. Tombol Back Mengambang (Floating):** Menghapus tombol back standar di Top Bar Detail dan menggantinya dengan tombol melayang estetik di pojok kiri atas gambar, sesuai deskripsi Anda.  
**4. Arsitektur Database Relasional:** Saya telah memperbarui skema database Room agar mendukung penyimpanan data taksonomi yang lebih kompleks secara permanen.  
**5. Multi-Language & Real-time Search:** Fitur ini sudah terintegrasi penuh dan responsif terhadap perubahan bahasa sistem maupun input pencarian Anda.  

# Tech Stack & Arsitektur
**• IDE:** Android Studio  
**• Language:** Java (JDK 11)  
**• Database:** Room Persistence Library dan EndemikDB  
**• Image Loading:** Glide  
**• UI/UX Components:** MaterialToolbar, BottomNavigationView, MaterialCardView, MaterialButton, ConstraintLayout, NestedScrollView, LinearLayout, RecyclerView dengan GridLayoutManager, ShapeableImageView
